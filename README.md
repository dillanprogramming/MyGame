# MyGame
What is MyGame?
MyGame is a text-based openworld game with Quest
how do i can activate it after a download?
create a text file and name it to boot.txt then write your ComputerUsername in and drag it to C:\Users\
after this you can finally start the game with a Double Click on it.
and if you will disable the input for the tutorial in openworld-0.04 the create another text file
and name it tutorial.txt and write False in, the drag it into C:\Users\
